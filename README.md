
# 🌱 Sensor de Umidade do Solo

**Aluno:** David Lucas  
**Turma:** DS3T  

## 📌 Sobre o Projeto

Este projeto demonstra o funcionamento básico de um **sensor de umidade**, utilizado para detectar a quantidade de água presente na terra. Esse tipo de sensor é muito utilizado em **sistemas de irrigação automática**, **agricultura inteligente (IoT)** e **monitoramento de plantas**.

## 🧠 Como o Circuito Funciona

1. O sensor é conectado ao **Arduino**.
2. O Arduino recebe os dados de umidade através de uma **porta analógica ou digital**.
3. O sistema analisa o valor recebido.
4. Dependendo da umidade do solo, o sistema acende o LED respectivo da situação
